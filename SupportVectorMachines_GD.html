Multiclass SVM Loss (hinge loss)¶

The SVM loss objective is to have a higher score on correct classes than the incorrect classes by some fixed margin $\Delta$.

The Multiclass SVM loss for the $i_{th}$ example is as follows:

$$ L_i = \sum_{j \neq y_i} max(0, x_i w_{j}^T - x_i w_{y_i}^T + \Delta) $$

If we chose example $i=0$ and assume that $w=(k,n)$, we see that target $y_0=1$, then $L_0=max(0, x_0 w_{0}^T - x_0 w_1^T + \Delta) + max(0, x_0 w_2^T - x_0 w_1^T + \Delta)$

Source: notebook is based on next link https://cs231n.github.io/linear-classify/#svm

def Li(x, y, W, delta=1.0):

    scores = np.dot(W, x)
    margin_def = scores - scores[y] + delta
    margins = np.maximum(0, margin_def)
    margins[y] = 0

    return np.sum(margins)

def margins(X, y, W, delta=1.0):
    m = X.shape[0]
    
    z = X.dot(W.T) # Compute the scores, (m,k)

    # get the score of the correct class for each example
    z_y = z[np.arange(m), y] #shape (m,)

    # Compute the margin by getting the max between 0 and the computed expression
    margin = np.maximum(0, z - z_y.reshape(-1, 1) + delta) #z_y[:, np.newaxis] could be used also
    margin[np.arange(m), y] = 0 #Remove delta from correct class
    
    return margin

2. Cost function¶

The loss function wants the score of the correct class $y_i$ to be larger than the incorrect class scores by at least by $\Delta$. If this is not the case, it will accumulate loss.

So, the objective is to find the weights that will simultaneously satisfy the constraint for all examples in the training data and give a total loss that is as low as possible.

We will use the most common regularization penalty (L2 norm) that discourages large weights through an elementwise quadratic penalty over all parameters as we have seen previously in other algorithms:

$$R(W) = \sum_k \sum_n W_{kn}^2$$

Including the regularization penalty completes the full Multiclass Support Vector Machine loss, which is made up of two components: the data loss and the regularization loss.

That is, the full Multiclass SVM cost function becomes: $L = \frac{1}{m} \sum_i L_i + \lambda R(W)$

Now, implement cost function $L$ in next cell, we will use the helper function margins:

In [8]:
def L(X, y, W, lmbda=0., delta=1.0):
    """
    - X: training examples of shape (m, n)
    - y: is array of integers, target
    - W: are the weights (k, n), k classes and n features
    """
    # number of examples
    m = X.shape[0]

    # margin 
    margin = margins(X, y, W, delta) # call margins

    # 1. Add all the losses together from margin
    loss = margin.sum()

    # 2. Divide the loss all over the number of training examples
    loss = loss/X.shape[0]

    # 3. Compute Regularization R(W) and add it to loss
    # loss = loss * lmbda*((W.sum())) # Correct form
    loss = loss + lmbda*(np.sum(W))
    
    return loss

Gradient of loss function¶

Recall that making good predictions on the training set is equivalent to minimizing the loss.

All we have to do now is to find the weights that minimize the loss; so, you are right, gradient descent.

$$ \nabla_{w_{y_i}} L_i = - \big(\sum_{j \neq y_i} 1(x_i w_{j}^T - x_i w_{y_i}^T + \Delta > 0) \big)x_i $$$$ \nabla_{w_{j}} L_i = 1\big(x_i w_{j}^T - x_i w_{y_i}^T + \Delta > 0\big) x_i $$

Function $1(expr)$ means that output will be $1$ when $expr$ is true.

For the equations, it only counts the number of classes that do not pass through the margin. For every example, it sum all the incorrect classes that was computed, and $x_i$ is scaled by that number.

Do not forget the regularization term.

In [10]:
k = w.shape[0] # classes
m = x.shape[0] # examples
delta = 1.0
loss = 0.0
dW = np.zeros(w.shape)
lmbda = 0.0

for i in range(m):
    # For each training example, we will count the scores and keep track of the correct class score
    z = x[i].dot(w.T)   # compute the score of x_i
    zy = z[y[i]]        # get z_y

    for c in range(k):
        if c == y[i]:   # skip when class index is the same as target
            continue
        
        margin = z[c] - zy + delta # Compute the margin

        if margin > 0:
            # the class contributes to the loss. And the gradient is computed
            loss += margin
            dW[y[i], :] -= x[i, :] # substract x[i], dW[y] - x_i
            dW[c, :] += x[i,:] # add x_i to class corresponding class index
            
dW /= m # divide by number of examples
dW += lmbda * 2 * w # Adding regularization

def gradient(X, y, W, lmbda, delta=1.0):
    # number of samples
    m = X.shape[0]
    # call helper function margins 
    margin = margins(X, y, W, delta)
    # 1. Create a zero array of same shape as margins
    mask = np.zeros((margin.shape))
    # 2. Set to 1 where margin values > 0
    mask[margin > 0] = 1
    # 3. Count the number of these examples where margin > 0, using mask
    count = np.sum(mask, axis = 1, dtype=int)
    # 4. Assign "-count" value to corresponding target positions in mask for each row, classes on target 'y'
    mask[np.arange(m), y] = -count.T
    # 5. Compute dot product of X.T and mask
    dW = np.dot(X.T, mask)
    # 6. Divide the gradient by the number of training examples
    dW /= m
    # 7. add the regularization derivative term
    dW += lmbda * np.sum(w.T) # Adding regularization
    
    return dW.T

def svm(x, y, alpha, lmbda, epochs):
    np.random.seed(2)
    
    n = x.shape[1]        # number of features
    k = len(np.unique(y)) # number of classes
    W = np.random.randn(k, n) # initialize your W
    
    J = {}
    for i in range(epochs):
        J[i] = L(x, y, W, lmbda, delta=1.0)      # compute the cost with current weights
        grad = gradient(x, y, W, lmbda, delta=1.0)  # get gradients
        
        # Now, update weights
        W = W - alpha * grad
        
    return W, J

def predict(x, w):
    return np.argmax(np.dot(x, w.T), axis=1)

def accuracy(x, y, w):
    y_hat = predict(x, w)
    return sum(y_hat == y)/len(y)

def gradient(X, y, W, lmbda, delta=1.0):
    # number of samples
    m = X.shape[0]
    # call helper function margins 
    margin = margins(X, y, W, delta)
    # 1. Create a zero array of same shape as margins
    mask = np.zeros((margin.shape))
    # 2. Set to 1 where margin values > 0
    mask[margin > 0] = 1
    # 3. Count the number of these examples where margin > 0, using mask
    count = np.sum(mask, axis = 1, dtype=int)
    # 4. Assign "-count" value to corresponding target positions in mask for each row, classes on target 'y'
    mask[np.arange(m), y] = -count.T
    # 5. Compute dot product of X.T and mask
    dW = np.dot(X.T, mask)
    # 6. Divide the gradient by the number of training examples
    dW /= m
    # 7. add the regularization derivative term
    dW += lmbda * np.sum((w**2).T) # Adding regularization
    
    return dW.T

def svm(x, y, alpha, lmbda, epochs, W):      
    J = {}
    for i in range(epochs):
        J[i] = L(x, y, W, lmbda, delta=1.0)      # compute the cost with current weights
        grad = gradient(x, y, W, lmbda, delta=1.0)  # get gradients
        
        # Now, update weights
        W = W - alpha * grad
        
    return W, J
