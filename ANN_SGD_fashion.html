1. Implementing activation functions¶

Sigmoid and tanh were monotonous, differentiable, and more popular activation functions. However, these functions suffer saturation over time, and this leads to problems occurring with vanishing gradients.

When classifying more than two classes, it is standard to use a softmax as output layer.

$$\sigma(z_k) = \frac{e^{z_k - max(z_k)}}{\sum_{j}^{K} e^{z_j - max(z_j)}}$$

For hidden layers, we will use ReLU and tanh:

Rectified Linear Unit (ReLU) is defined as $f(z)=max(0, z)$

The derivative is: $ f'(z) = \begin{cases} 1, & \quad \text{if } z > 0\\ 0, & \quad \text{if } \text{otherwise} \end{cases} $

Implement sigma and d_sigma functions with three activation option:

  1. softmax
  2. tanh
  3. relu

You should also take a look to Leaky ReLU and Parameterised ReLU.

In [7]:
def softmax(p_z):
    """
    p_z -> x*w.T = (m,k)
            
    $$\sigma(z_k) = \frac{e^{z_k - max(z_k)}}{\sum_{j}^{K} e^{z_j - max(z_j)}}$$

    """  
    
    expZ = np.exp(p_z - np.max(p_z)).T 
    return (expZ / expZ.sum(axis=0, keepdims=True)).T


def sigma(p_z, p_act):
    """
    
    p_z: 

    p_act:

    """

    if p_act == 'softmax':
        return softmax(p_z)

    elif p_act == 'relu':
        return max(0, p_z)

    elif p_act == 'tanh':
        return np.tanh(p_z)


def d_sigma(p_a, p_act):
    """
    
    p_a

    p_act

    """

    if p_act == 'softmax':
        
        s = sigma(p_z, 'softmax')
        s = s.reshape(-1,1)

        return np.diagflat(s) - np.dot(s, s.T)

    elif p_act == 'tanh':

        a = sigma(p_z=p_a, p_act='tanh')

        return 1 - a**2
    
    elif p_act == 'relu':

        return 1 if p_z > 0 else 0

def accuracy(A, y):

    true = y.astype(int)
    pred = predict(A)
    classes = len(np.unique(true))
    cm = np.zeros((classes, classes))

    for i in range(len(true)):
        cm[true[i]][pred[i]] += 1

    return np.sum(np.diag(cm))/y.shape[0]

3. Xavier initialization¶

Their major goal is to prevent gradient vanishing and too-large weight problems, the formula for a normal distribution is as follows:

$$ W_l = randn(n_l, n_{l-1}) * \sqrt{2 / (n_{l-1} + n_l)} $$$$ \textbf{Xavier in (glorot and bengio, 2010) ?} \quad \rightarrow \quad W \sim U \left[ - \frac{\sqrt{6}}{\sqrt{n_{j} + n_{j+1}}} , \frac{\sqrt{6}}{\sqrt{n_{j} + n_{j+1}}} \right] \qquad (16) $$

Note: for each $l$ layer, $n_{l-1}$ is the number of input neurons and $n_l$ is output neurons.

In [11]:
def Xavier_init_w(nx, nh, ny):
    """
    nx, number of features
    nh, number of hidden neurons
    ny, number of output neurons
    """

    np.random.seed(1)
    
    # w's will be created randomly
    W1 = np.random.randn(nh, nx) * np.sqrt(2/(nx + nh))
    W2 = np.random.randn(ny, nh) * np.sqrt(2/(nh + ny))
    
    # b's will be zeros
    b1 = np.zeros((nh, 1))
    b2 = np.zeros((ny, 1))
    
    W = {"W1": W1, "b1": b1, "W2": W2, "b2": b2}
    
    return W

4. Cost function¶

Cross entropy error function for multi-class output is:

$$ E = - \sum_{k}^{K} y_k log(a_k) $$

Where:

$$ a_k = \sigma_k(z_k) = \frac{e^{z_k}}{\sum_{j}^{K}e^{z_{j}}} $$

Then, the cost for all examples:

$$ J(w) = \frac{1}{m}\sum_{i=1}^{m} E_i $$

Regularization is as follows:

$$ J(w) = \frac{1}{m}\sum_{i=1}^{m} E_i + \frac{\lambda}{2m} \sum_{l=1}^{L-1} \sum_{i=1}^{n_{l+1}} \sum_{j=1}^{n_l} w_{lji}^2 $$

Note:

  1. Implement cost function without regularization term
  2. We are NOT going to use one-hot target

def cost(A, y):
    
    y_hat = np.zeros(shape=(y.shape[0], A.shape[1]))
    y_hat[range(len(y)), y] = 1
    return (np.sum(np.sum((-y_hat * np.log(A)))))/y_hat.shape[0]

Backpropagation algorithm¶

5. Feed-forward in a vectorized form¶

  1. Compute the weighted sum $z_{l+1} = a_{l} w_{l}^T + b_{l}^T$
  2. Apply the activation functions $a_{l+1} = \sigma(z_{l+1})$

Implement forward propagation to calculate output probabilities

In [16]:
def forward(x_f, w_f, a_f):
    """
    activation: for hidden layer
    """

    # hidden layer
    Z2 = np.matmul(x_f, w_f['W1'].T) + w_f['b1'].T 
    A2 = sigma(Z2, a_f)

    # output layer
    Z3 = np.dot(A2,  w_f['W2'].T) + w_f['b2'].T
    A3 = sigma(Z3, 'softmax')
    
    # results
    Z = {"Z2": Z2, "A2": A2, "Z3": Z3, "A3": A3}
    
    return A3, Z

6. Backward-propagation¶

$ \frac{\partial{E_i}}{\partial z_k} = \begin{cases} a_k - 1(y_i = k) &\text{if $y_i$ is the class}\\ a - y &\text{if $y$ is one-hot} \end{cases}$ \end{equation*}

Where,

$ (y=k)= \begin{ases} 1 & \text{if y=k}\\ 0 & \text{otherwise} \end{cases} $

  1. For output layer $\delta_l = \frac{1}{m} a - y$
  2. For hidden layer $\delta_l = \sigma'(z_l) * \delta_{l+1} w_l$

Gradients

  1. For bias term, $\frac{\partial}{\partial b_l} = sum(\delta_{l+1})$, recall that shape of $\frac{\partial}{\partial b_l}$ must match the shape of $b_l$
  2. For weights, $\frac{\partial}{\partial w_l} = \delta_{l+1} a_l$

Derivative of the regularization term is as follows: $\frac{\partial}{\partial w_l} = \frac{\lambda}{m} w_l$

So, $\frac{\partial}{\partial w_l} = \delta_{l+1} a_l + \frac{\lambda}{m} w_l$

In [18]:
def backward(w, z, x, y, activation):
    """
    activation: for hidden layer
    """
    m = x.shape[0]
    A1 = x
    
    # weights for neurons and biases in input layer
    W1 = w['W1']
    b1 = w['b1']

    # weights for neurons and biases in hidden layer
    W2 = w['W2']
    b2 = w['b2']

    # weighted-inputs output and activation output of hidden layer
    A2 = z['A2']
    Z2 = z['Z2']
    
    # weighted-inputs output and activation output of output layer
    A3 = z['A3']
    Z3 = z['Z3']
        
    # Backward propagation: calculate dW1, db1, dW2, db2.
    # display(A3.shape) 16800, 4
    
    y_hat = np.zeros(shape=(len(y), A3.shape[1]))
    y_hat[range(len(y)), y] = 1

    d3 = (1/m)*(A3 - y_hat)
    
    dW2 = np.dot(d3.T, A2)
    db2 = np.sum(d3)

    d2 = d_sigma(A2, activation)*np.dot(d3, W2)
    dW1 = np.dot(d2.T, A1)
    db1 = np.sum(d2)
    
    return {"dW1": dW1, "db1": db1, "dW2": dW2, "db2": db2, 'd2': d2, 'd3':d3}

Implement the model function:

In [19]:
def model(x, y, nh, alpha, lmbda, epochs, activation):
    m, nx = x.shape
    ny = 4 # output neurons
    
    # initialize Weights
    W = Xavier_init_w(nx, nh, ny)
    
    A3, Z = forward(x_f=x, w_f=W, a_f=activation)          # feed-forward
    print('Initial cost:', cost(A3, y))                    # print initial cost
    
    J = {}
    for i in range(epochs):
        A3, Z = forward(x_f=x, w_f=W, a_f=activation)
        J[i] = cost(A3, y)
        # print('cost in ', i, ' is: ', J[i])
        grad = backward(W, Z, x, y, activation)

        # Now, update weights

        # -- Olascuaga regularization (according to formulas in this notebook)
        # O = + (lmbda/m)*W['W1']

        # -- L1 (LASSO)
        # L1 = + (lmbda/m)*np.sum(abs(W['W1']))

        # -- L2 (RIDGE)
        # L2 = + (lmbda/m)*np.sum(W['W1']**2)
        
        W['W1'] = W['W1'] - alpha*grad['dW1'] - (lmbda/m)*W['W1']
        W['b1'] = W['b1'] - alpha*grad['db1']

        W['W2'] = W['W2'] - alpha*grad['dW2'] - (lmbda/m)*W['W2']
        W['b2'] = W['b2'] - alpha*grad['db2']
    
    print('Final cost:', J[epochs-1])
    
    return W, J

7. Stochastic Gradient Descent¶

This is an extension of the normal gradient-descent algorithm. Thus, if we have $m$ training examples, the gradient function will also have $m$ additive terms.

SGD depends on a simple insight that the gradient is actually an expectation. We can approximate the expectation by computing it on small sample sets.

Now, instead of using all m training examples to calculate the derivative, we will take only one example, with this change we can still reasonably approximate the derivative.

def SGD(x, y, nh, alpha, lmbda, epochs, activation, debug=False):
    
    # np.random.seed(2)
    m, nx = x.shape
    ny = 4 # output neurons   
    W = Xavier_init_w(nx, nh, ny) # init weights
    J = {}

    for it in range(epochs):

        # feed-forward
        A3, Z = forward(x_f=x, w_f=W, a_f=activation)

        # cost
        J[it] = cost(A3, y)

        # print(J[it])
                
        # shuffle the indices for each epoch
        idxs = np.array(range(m))
        np.random.shuffle(idxs)
       
        for i in idxs:

            # get one image (1, nx)
            x1 = np.array(x[i].reshape(1, nx))
            y1 = np.array([y[i]])

            A3, Z = forward(x_f=x1, w_f=W, a_f=activation)  # forward 
            grad = backward(W, Z, x1, y1, activation)       # backward
            
            # Now, update weights
            W['W1'] = W['W1'] - alpha*grad['dW1'] - (lmbda/m)*W['W1']
            W['b1'] = W['b1'] - alpha*grad['db1']

            W['W2'] = W['W2'] - alpha*grad['dW2'] - (lmbda/m)*W['W2']
            W['b2'] = W['b2'] - alpha*grad['db2']
       
        if it % 10 == 0 and debug:
            print('epoch', it, 'cost', J[it])
    
    print('Final cost:', J[epochs-1])

    return W, J

8. SGD with batches¶

Now, instead of using only one training example to calculate the derivative, we will take a small random sample of the training set, we can have a better approximation on the derivative.

The minibatch function, instead of return keyword, we are using yield.

yield is a keyword that is used like return, except the function will return a generator.

In [57]:
def minibatch(x, y, size):
    np.random.seed(2)
    m, n = x.shape
    idxs = np.arange(m)
    np.random.shuffle(idxs)
    for idx in range(0, m - size + 1, size):
        batch = idxs[idx: idx + size]
        yield x[batch], y[batch]

def SGD_batch(x, y, nh, alpha, lmbda, epochs, activation, batch_size, debug=False):
    m, nx = x.shape
    ny = 4
    
    W = Xavier_init_w(nx, nh, ny) # init weights 

    J = {}   
    for it in range(epochs):
        
        A3, Z = forward(x_f=x, w_f=W, a_f=activation)  # forward
        J[it] = cost(A3, y) # cost
        
        for batch in minibatch(x, y, size = batch_size): #use minibatch here
            a1, y_batch = batch
            y1 = y_batch

            A3, Z = forward(x_f=a1, w_f=W, a_f=activation)  # forward
            grad = backward(W, Z, a1, y1, activation)  # backward
            
            # Now, update weights
            W['W1'] = W['W1'] - alpha*grad['dW1'] - (lmbda/batch_size)*W['W1']
            W['b1'] = W['b1'] - alpha*grad['db1']

            W['W2'] = W['W2'] - alpha*grad['dW2'] - (lmbda/batch_size)*W['W2']
            W['b2'] = W['b2'] - alpha*grad['db2']
        
        if it % 10 == 0 and debug == True:
            print('epoch', it, 'cost', J[it])     
    
    print('Final cost:', J[epochs-1])

    return W, J

def confusion_matrix(A, y):
    k = A.shape[1]
    mat = np.zeros((k, k))
    y_hat = predict(A)
    for i in range(k):
        T = (y_hat == i) #all equal to class k
        for j in range(k):
            mat[i, j] = sum(y[T] == j)
    return mat

def plot_confusion_matrix(values, classes):
    import itertools
    m, n = values.shape

    accuracy = np.trace(values) / float(np.sum(values))
    misclass = 1 - accuracy

    plt.figure(figsize=(8, 6))
    plt.imshow(values, interpolation='nearest', cmap=plt.get_cmap('Blues'))
    plt.title('Confusion matrix')
    plt.colorbar()

    marks = np.arange(len(classes))
    plt.xticks(marks, classes, rotation=45)
    plt.yticks(marks, classes)

    thresh = values.max() / 2
    for i, j in itertools.product(range(m), range(n)):
        plt.text(j, i, "{:,}".format(values[i, j]),
                 horizontalalignment="center",
                 color="white" if values[i, j] > thresh else "black")

    plt.tight_layout()
    plt.ylabel('Predicted label')
    plt.xlabel('True label\naccuracy={:0.4f}; misclass={:0.4f}'.format(accuracy, misclass))
    plt.show()
